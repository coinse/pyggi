pyggi.logger module¶

This module contains Logger class.

class pyggi.logger.Logger(name)[source]¶

Bases: object

Logger class is used to records execution information. It wraps the Logger object from logging module which have two handlers: file handler and stream handler. 5 logging levels are available: debug, info, warning, error, critial. For more information, see https://docs.python.org/3.6/library/logging.html .
